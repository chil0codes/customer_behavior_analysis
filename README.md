# Data Analytics Project

## Overview
This project demonstrates an end-to-end data analytics workflow, from data extraction and cleaning to visualization and reporting. The objective is to transform raw data into meaningful insights that support data-driven decision-making. The project includes data analysis in Python, SQL querying, dashboard development in Power BI, and presentation of findings through a report and presentation slides.

## Dataset
The dataset used in this project contains information relevant to the business problem being analyzed. It was imported into Python for exploration, cleaned to improve data quality, and loaded into a relational database for SQL analysis.

**Key activities:**
* Data loading and inspection
* Handling missing values
* Removing duplicates
* Correcting data types
* Feature exploration and analysis

## Project Workflow
### 1. Data Loading
* Imported the dataset into Python.
* Reviewed dataset structure, dimensions, and data types.

### 2. Exploratory Data Analysis (EDA)
* Generated summary statistics.
* Identified trends, patterns, and anomalies.
* Visualized key metrics using charts and graphs.

### 3. Data Cleaning
* Addressed missing values.
* Removed duplicate records.
* Standardized data formats.
* Prepared data for analysis and reporting.

### 4. SQL Analysis
* Loaded cleaned data into a relational database.
* Wrote SQL queries to:
  * Filter and aggregate data
  * Calculate KPIs
  * Identify trends and patterns
  * Generate analytical summaries

### 5. Power BI Dashboard
* Connected Power BI to the processed dataset.
* Built interactive dashboards and visual reports.
* Created filters and slicers for dynamic analysis.

### 6. Reporting & Presentation
* Documented findings in a business report.
* Created a presentation in Gamma to communicate key insights and recommendations.

## How to Run
### Prerequisites
* Python 3.x
* PostgreSQL / MySQL / SQL Server
* Power BI Desktop

### Steps
1. Clone the repository:
```bash
git clone <repository-url>
```

2. Install required Python packages:
```bash
pip install pandas numpy matplotlib seaborn
```

3. Open and run the Python notebook/script for data cleaning and EDA.

4. Load the cleaned dataset into your SQL database.

5. Execute the SQL queries provided in the `sql/` folder.

6. Open the Power BI file (`.pbix`) and refresh the data connection.

7. Review the dashboard, report, and presentation materials.
